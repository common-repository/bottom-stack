== Description ==

Bottom Stack is used to display the extra text from the stack. You have 8 text boxes into which you can save your text, HTML or PHP code and display it anywhere on static pages and blog posts.

You can specify whether the data is automatically displayed at the bottom on every static page or blog post.

Plugin is useful for displaying variable signatures, forms, adverts etc.. and change the data content on the one place.

== Installation ==

1.	Unzip archive to the '/wp-content/plugins/' directory
2.	Activate the plugin through the 'Plugins' menu in WordPress
3.	Adjust your text boxes in the 'Settings' menu under the name 'Bottom Stack'

**Usage**
You can add a data stack directly to your page or post by inserting the trigger text [btmstack1], [btmstack2] ...
or set displaying on each of them by option beside the textarea.
